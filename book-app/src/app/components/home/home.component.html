Welcome to BOOK APP

Book App is Single Page Web Application powered by Angular, Spring Boot Microservices and polyglot persistence with MongoDB along with MySQL databases running on docker containers.

Book App allows users to search and retrieve information about books using Google Book API, and add their favourite books to a personal list. User can also recommend books to other users so that they find something good to read when the choice is humongous.
